# ClearWord
A simple (Local) text editor with no tracking or telemetry. Russian and English language support. Please read before using README.md

# RU:

# Возможности
- Собственное расширение .clew
- Сохранение настроек каждого документа с помощью конфигурации. Конфиг появляется в папке ClearConfig в подпапке ClearWord, его удалять нельзя!
- Переключение языков
- Проверка орфографии (вкл/выкл)
- Изменение цветов текста и фона
- Изменение стиля текста - Жирный/Курсив/Подчёркнутый
- Отображение счётчика слов и позиции курсора
- Отображение несохранённых изменений
- Единый шрифт Clear для всего документа

# Установка

Скачай ClearWord.exe из [Releases] (https://github.com/ClearGroups/ClearWord/releases) и запусти.

# EN:

# Features
- Own extension .clew
- Saving the settings of each document using configuration. The config appears in the ClearConfig folder in the ClearWord subfolder, it cannot be deleted!
- Language switching
- Spell check (on/off)
- Change text and background colors
- Change the text style - Bold/Italics/Underlined
- Displaying the word counter and cursor position
- Display of unsaved changes
- A single Clear font for the entire document

# Installation

Download it ClearWord.exe from [Releases] (https://github.com/ClearGroups/ClearWord/releases ) and launch it.
